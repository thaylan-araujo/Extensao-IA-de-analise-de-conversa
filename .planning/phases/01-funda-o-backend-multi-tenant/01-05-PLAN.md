## Trust Boundaries

| Boundary | Description |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-16 | Spoofing | Token de convite adivinhável/reutilizável | mitigate | 32 bytes aleatórios; só SHA-256 no banco; single-use (status accepted); expiração 7d; reenvio invalida o anterior |
| T-1-17 | Elevation of Privilege | Advogado (ou anônimo) cria convites | mitigate | Handler exige profile gestor/super_admin ativo antes de tocar o createAdminClient; teste cobre 401/403 |
| T-1-18 | Elevation of Privilege | Convite forjado com role super_admin | mitigate | Zod restringe a advogado/gestor + CHECK no schema (defesa dupla) |
| T-1-19 | Denial of Service | Spam de convites por conta de gestor comprometida | mitigate | Teto de 20 convites pending por organização (429) |
| T-1-20 | Information Disclosure | Accept revela por que o token falhou (enumeração) | mitigate | Resposta única 410 "Convite inválido ou expirado." |
| T-1-21 | Tampering | Aceite duplo cria conta duplicada nas métricas | mitigate | Reativação D-12 reaproveita o usuário existente; D-08 bloqueia segundo vínculo com 409 |
