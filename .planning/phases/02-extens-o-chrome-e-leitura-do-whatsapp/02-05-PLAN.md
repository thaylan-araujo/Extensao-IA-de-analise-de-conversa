## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Gestor/advogado → /api/admin/settings | Usuário comum não pode acionar o kill-switch global |
| Toggle → app_settings | Mutação de flag que afeta todos os tenants de uma vez |
| reader_status → tela admin | Dados operacionais agregados de todos os tenants (só super-admin) |

## STRIDE Threat Register

| T-02-17 | Repudiation | acionamento do kill-switch | mitigate | logAudit obrigatório com action reader.kill_switch.toggled e actor — padrão de auditoria da Fase 1 |
