Policies: `"authenticated read settings"` FOR SELECT TO authenticated USING (true); `"super admins manage settings"` FOR ALL TO authenticated USING/WITH CHECK `(select private.is_super_admin())`. Índice `reader_status_organization_id_idx`. RLS habilitado. Policies: SELECT para a própria linha (`profile_id = (select auth.uid())`) OU `(select private.is_super_admin())` (D-14: só a Elite Juris agrega); INSERT e UPDATE TO authenticated com USING/WITH CHECK `profile_id = (select auth.uid()) and organization_id = (select private.current_org_id())`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Extensão (anon key + sessão advogado) → Postgres | Toda escrita de conversas/mensagens cruza RLS |
| Advogado → app_settings | Usuário comum não pode desligar/ligar a leitura global |
| Advogado → reader_status | Heartbeat só da própria linha; leitura agregada só super-admin |
